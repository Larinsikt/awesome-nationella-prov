# Bidra till Awesome Nationella Prov

Tack för att du vill bidra. Den här listan blir bara så bra som dess bidragsgivare.

## Hur du bidrar

1. Forka det här repot
2. Skapa en branch: `git checkout -b add-resurs-namn`
3. Lägg till din resurs i rätt sektion av `README.md`
4. Skicka en pull request med kort motivering

För mindre ändringar (stavfel, dödlänkar) räcker det att öppna en [issue](../../issues/new).

## Riktlinjer

För att en resurs ska tas in i listan ska den:

- Vara relevant för svenska nationella prov
- Vara svenskspråkig eller ha tydlig svensk kontext
- Vara fritt tillgänglig eller ha en gratisversion (kommersiella verktyg får finnas, max ett per kategori)
- Ha ett aktivt underhåll (inga övergivna sajter)
- Inte vara en affiliate- eller hänvisningslänk

Skolverkets officiella resurser har företräde i alla kategorier.

## Format

Använd den här mallen för varje resurs:

```markdown
- [Resursens namn](https://exempel.se) – kort beskrivning på max en mening
```

- Beskrivningen ska börja med stor bokstav
- Skriv naturligt, undvik säljspråk
- Inga emojis i resursrader

## Kategorier

Lägg resursen i den mest specifika sektion som passar. Är du osäker, lägg den i "Verktyg och appar" och nämn det i din PR.

## Granskning

Pull requests granskas av Alexander Johansson (legitimerad lärare). Räkna med svar inom 1–2 veckor. Vid avslag förklarar vi varför.

## Code of conduct

Var respektfull. Diskutera resurser, inte personer. Vi följer [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
